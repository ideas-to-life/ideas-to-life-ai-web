# Runner Agentic Intelligence

## Why This Exists
Complex tasks often require long-running, stateful interactions that simple request-response models cannot handle.

## Features
**Runner** is our production-grade framework for agentic intelligence. It allows for the reliable execution of complex, multi-step tasks that require reasoning, tool use, and error recovery.

- **Reliability**: Built for long-running processes.
- **Observability**: Detailed logs and traces of agent thought processes.

We are **Shipping** this framework to select partners for real-world automation tasks.
